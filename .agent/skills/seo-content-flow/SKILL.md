# seo-content-flow: SEO記事制作フロー

SEOを意識した記事を設計・執筆・公開するフローを标准化する。

## フロー

```
1. キーワード選定
   - 検索ボリューム・競合難易度の確認
   - LSIキーワードの抽出

2. 構成設計
   - H1〜H3の見出し構成を先に作る
   - 想定読者・検索意図を明記する

3. 執筆
   - humanizer で AI文を自然化する
   - 一次情報・実績データを盛り込む

4. SEOチェック
   - seo-checklist を実行する

5. 公開・計測
   - Google Search Console で登録確認
   - 2週間後に順位を確認する
```
