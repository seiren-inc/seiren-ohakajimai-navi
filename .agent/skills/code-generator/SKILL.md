# code-generator: コード生成

要件定義からプロダクション品質のコードを生成する。

## 生成ルール

- TypeScript / React / Next.js を標準とする
- 型定義を必ず付ける（`any` 禁止）
- コメントは日本語で記述する
- エラーハンドリングを必ず実装する

## チェックリスト

- [ ] 型定義が完全か
- [ ] エラーハンドリングがあるか
- [ ] テストが書けるコード構造か
- [ ] 命名規則が一貫しているか
