# security-audit: セキュリティ監査

アプリケーションのセキュリティ脆弱性を検出する。

## チェックリスト

### 認証・認可
- [ ] JWTの署名検証が実装されているか
- [ ] セッションタイムアウトが設定されているか
- [ ] 管理者機能のロールチェックがあるか

### インジェクション
- [ ] SQLインジェクション対策（プリペアドステートメント使用）
- [ ] XSSへの対策（出力エスケープ）
- [ ] CSRFトークンが実装されているか

### データ保護
- [ ] 機密データが暗号化されているか
- [ ] ログに機密情報が含まれていないか
- [ ] HTTPS強制が設定されているか

### 依存関係
- [ ] `npm audit` でHigh/Critical脆弱性がないか
