# summarize

## When to Use
- User shares a URL and wants key takeaways
- Summarizing a PDF or long document
- Extracting key points from research or articles
- Competitive analysis of competitor pages
- Condensing meeting notes or reports

## Output Format
```
## Summary: [Title]

**What it is**: [1 sentence]

**Key Points**:
- Point 1
- Point 2
- Point 3

**Takeaway**: [1-2 sentences on what matters]
```

## Tips
- For Japanese content: summarize in Japanese
- For technical docs: preserve code examples and version numbers
- For marketing pages: note pricing, features, target audience
- Keep summaries under 300 words unless detail is requested
