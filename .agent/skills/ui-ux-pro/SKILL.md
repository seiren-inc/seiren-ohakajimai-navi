# ui-ux-pro: UX・CV最適化

ユーザー体験と成果指標（CV・滞在時間・直帰率）を改善する。

## CVO観点チェック

- [ ] CTAが画面内に常に見えるか
- [ ] フォームのフィールド数が最小限か
- [ ] 読み込み速度が3秒以内か
- [ ] モバイルでのタップターゲットが44px以上か
- [ ] エラーメッセージが具体的か

## ファーストビュー設計

1. H1でベネフィットを伝える（機能ではなく）
2. 社会的証明（実績数・クライアント名）
3. CTA（主と副の2種）
