# using-superpowers: Skill使用の最上位ルール

全タスクで以下を必須とする。

## 必須プロセス

1. **Skill探索**: タスク開始前に `find-skills` を呼び出し最適なSkillを特定する
2. **Skill呼び出し**: 該当SKILL.md を読み込み、指示に従って実行する
3. **チェックリスト実行**: Skill内のチェックリストを全項目実行する
4. **完了報告**: 使用したSkillと実行結果を報告する

## 適用ルール

- Skillなしでタスクを開始することを禁止する
- 該当Skillが存在しない場合は `skill-manager` で新規作成を提案する
- 複数のSkillが該当する場合は全て読み込んで統合する

## 例外

- 単純な質問応答（コード生成・設計・テスト・デプロイは例外対象外）
