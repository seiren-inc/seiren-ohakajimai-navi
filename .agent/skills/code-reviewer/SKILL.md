# code-reviewer: コードレビュー

コードの品質・セキュリティ・パフォーマンスを確認する。

## レビュー観点

### 必須チェック
- [ ] SQLインジェクション・XSSの脆弱性がないか
- [ ] 認証・認可が適切か
- [ ] 機密情報がハードコードされていないか
- [ ] エラーハンドリングが適切か

### 推奨チェック
- [ ] テストカバレッジが十分か
- [ ] コメントが適切か
- [ ] パフォーマンスの問題がないか
- [ ] 命名規則が一貫しているか
