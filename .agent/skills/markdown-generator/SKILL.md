# markdown-generator: Markdownドキュメント生成

技術ドキュメント・仕様書・READMEをMarkdown形式で生成する。

## テンプレート種別

| 種別 | 用途 |
|---|---|
| README | プロジェクト概要・セットアップ手順 |
| API仕様書 | エンドポイント一覧・リクエスト/レスポンス |
| 設計書 | システム構成・ER図・フロー図 |
| 手順書 | 作業手順・チェックリスト |

## 記述ルール

- 見出しは H2 から開始（H1 はタイトルのみ）
- コードブロックは言語を明記する
- テーブルは必ずヘッダーを付ける
