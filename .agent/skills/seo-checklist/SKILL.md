# seo-checklist: SEOチェックリスト

ページ公開前にSEO内部対策を確認する。

## メタ情報

- [ ] `<title>` が30〜35文字以内か（キーワードを含むか）
- [ ] `<meta description>` が70〜120文字か
- [ ] OGP（og:title / og:description / og:image）が設定されているか
- [ ] canonical URL が設定されているか

## コンテンツ

- [ ] H1が1つだけあるか
- [ ] H1にメインキーワードが含まれるか
- [ ] 画像に `alt` テキストがあるか
- [ ] 内部リンクが3件以上あるか

## テクニカル

- [ ] ページ速度が3秒以内か（Lighthouse スコア70以上）
- [ ] モバイルフレンドリーか（Google Search Console 確認）
- [ ] `robots.txt` にインデックスを禁止していないか
- [ ] sitemap.xml が最新の状態か
