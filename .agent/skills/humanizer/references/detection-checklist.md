# AI文章 検出チェックリスト

テキストレビュー時の簡易チェックリスト。該当3つ以上でHumanizer是正を実行。

## チェック項目
- [ ] 「画期的」「革新的」など誇大表現がある
- [ ] 「多くの」「様々な」など曖昧な表現がある
- [ ] 「心に寄り添う」など感情訴求ワードが根拠なく使われている
- [ ] 「〜といえるでしょう」がある
- [ ] 「重要です」が2回以上ある
- [ ] 「〜が求められます」がある
- [ ] 「〜することが可能です」がある（→「〜できます」に置換可）
- [ ] 三点セット（「A・B・C」）が2箇所以上ある
- [ ] 接続詞（さらに、加えて、また）が1段落に2つ以上ある
- [ ] 絵文字が業務テキストに含まれている
- [ ] 太字が3箇所以上ある
- [ ] 「いかがでしたでしょうか」がある
- [ ] ポエム的な結びの段落がある
- [ ] 「素晴らしいご質問ですね」系の追従表現がある

## 判定基準
| 該当数 | 判定 | アクション |
|:------:|:----:|-----------|
| 0-2 | 合格 | そのまま使用可 |
| 3-5 | 要修正 | 該当箇所のみ修正 |
| 6-10 | 全面修正 | Humanizerプロセスで全文書き換え |
| 11+ | 却下 | ゼロから書き直し |
