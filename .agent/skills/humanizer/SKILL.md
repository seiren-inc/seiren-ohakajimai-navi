# Humanizer: AI文章の自然化

AI生成テキストの「AI臭さ」を検出し、実務で使える自然な日本語に書き換える。

## 適用対象
- Webサイトのコピー（見出し・本文・CTA）
- LP / サービス説明
- 管理画面のUI文言
- ドキュメント・マニュアル

## 是正プロセス

1. **検出**: 30パターンでスキャンし該当箇所を特定
2. **初回修正**: 各パターンのルールに従い書き換え
3. **再監査**: 新たなAI臭さが発生していないか確認
4. **最終チェック**: 「実在する企業のWebサイトに掲載されていて違和感がないか」

## 業種別注意点

**清蓮グループ（葬祭・墓石）**: 格調は必要だが感情訴求系は禁止。事実と手続きを明確に。

**SaaS（SocialBoost等）**: 機能説明は「何ができるか」を簡潔に。「革新的」「画期的」は使わない。

**SEOメディア**: 「いかがでしたでしょうか」は絶対禁止。まとめ段落のポエム化を防ぐ。
