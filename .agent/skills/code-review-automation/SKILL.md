# code-review-automation: 自動コードレビュー

PRの差分を解析してコード品質・セキュリティ・パフォーマンスを自動チェックする。

## チェック観点

### セキュリティ
- [ ] `process.env` 以外での機密情報取得がないか
- [ ] SQL文の文字列結合がないか（SQLインジェクション）
- [ ] ユーザー入力の直接レンダリングがないか（XSS）

### パフォーマンス
- [ ] `useEffect` の依存配列が適切か
- [ ] 無限ループになるクエリがないか
- [ ] 大量データの全件取得がないか

### 保守性
- [ ] 関数が50行以内か
- [ ] コメントがない複雑なロジックがないか
